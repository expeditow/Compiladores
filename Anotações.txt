#10 Operadores Lógicos  
- Implementar os operadores lógicos `&&`, `||` e `!`. Deve-se verificar a compatibilidade de tipos nas expressões lógicas.

Código na LP:
bool B1;
bool B2;
bool R;
R = B1 && !B2;

Código Intermediário:
int T1;
int T2;
int T3;
int T4;
int T5;

T2 = !T1;
T4 = T3 && T2;
T5 = T4;
